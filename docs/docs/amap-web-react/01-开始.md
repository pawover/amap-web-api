# 开始

## 介绍

[amap-web-react](https://github.com/Handpear/amap-web-api) 是基于 React 封装的高德地图组件库。

> 建议搭配 [amap-web-types](/amap-web-types/01-开始) 一起使用

特性：

- ⌛ 通过创建 `script` 标签的形式自动加载高德地图 SDK
- 💫 使用 Typescript 编写，更友好的类型提示
- ⚛️ 支持 React Hooks

## 安装

::: code-group

```sh [pnpm]
pnpm add -D amap-web-react

```

:::
